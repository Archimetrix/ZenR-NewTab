## ✨ Features

| | Feature | Description |
|---|---|---|
| 🪟 | **Liquid Glass UI** | Apple-inspired glass-morphism with 11 live tuning controls |
| 📋 | **Pages & Boards** | Multi-page layout with draggable, color-tinted glass boards |
| 🖼️ | **Dynamic Wallpapers** | Gradient presets, image upload, and live video backgrounds |
| 🔍 | **Search Bar Widget** | 6 search engines with live suggestions and compact mode |
| 🕐 | **Clock & Date Widget** | 5 display styles, 12/24h, seconds toggle, size control |
| 🗒️ | **Notes Widget** | Floating draggable notepad with image support |
| 🧮 | **Calculator Widget** | Full calculator with optional history log |

| 🔒 | **100% Local** | No accounts, no tracking, no data collection |

---

## 🗂️ Layouts

### 🃏 Classic Boards
Draggable glass cards arranged in a free-form grid. Each board has its own name and optional color tint. Boards live inside named **Pages**, so you can keep Work, Entertainment, Research, and anything else cleanly separated.

### 🔲 Minimal Groups
A clean, distraction-free icon grid layout. Adjust links per row (3–16) and choose from three icon styles:

| Style | Look |
|---|---|
| Default | Solid favicon bubbles |
| Glass | Frosted glass icon chips |
| Minimal | Borderless favicon-only |

> **Note:** Links and widgets are independent between layouts and are not synced.

---

## 🪟 Liquid Glass Engine

Every card surface in ZenR uses a fully customizable glass renderer. All settings are live-editable from the Settings panel:

| Parameter | Range | Effect |
|---|---|---|
| Glass Blur | 0 – 72px | Frosted backdrop blur |
| Glass Density | 0 – 25% | Surface fill opacity |
| Saturation | 100 – 350% | Color vibrancy through glass |
| Brightness | 85 – 130% | Overall card brightness |
| Rim Glow | 0 – 100% | Inner edge light halo |
| Shadow Depth | 0 – 80% | Drop shadow intensity |
| Glass Tint | Color picker | Surface color overlay |
| Tint Intensity | 5 – 50% | Tint color strength |
| Corner Boost | 0 – 100% | Corner highlight effect |
| Edge Shimmer | 0 – 100% | Edge light animation |
| Refraction | 0 – 100% | Light bending distortion |

All values reset to defaults with one click.

## 🖼️ Wallpaper System

### 🎨 Gradients
One-click gradient presets — a curated palette always ready to go.

### 🏞️ Images
- **Pixabay Search** — Search millions of free images from inside ZenR using your own free Pixabay API key
- **Local Upload** — Upload JPG, PNG, WebP, or GIF files (multi-select supported)
- **Auto-rotate** — Get a fresh wallpaper on every new tab

### 🎬 Live Video Backgrounds
- **Pixabay Video Search** — Ambient video loops searchable directly inside ZenR
- **Local Upload** — Upload MP4, WebM, or OGV files
- **Auto-rotate** — Cycle your video collection on each tab open

### 🎛️ Background Controls
Fine-tune **overlay opacity** (0–90%) and **background blur** (0–30px) to keep content readable over any wallpaper.

## ⚙️ Widgets

### 🕐 Clock & Date
Five display styles to match any aesthetic:

| Style | Description |
|---|---|
| Glass Card | Frosted glass panel with soft shadow |
| Minimal | Clean text-only, no background |
| Neon Glow | Colored glow on digits |
| Retro Digital | Monospace dot-matrix look |
| Bold Centered | Large, full-width centered time |

Supports **12-hour / 24-hour** formats, optional **seconds display**, and a **size scale** control.

### 🗒️ Notes
A floating, draggable notepad that persists across sessions. Supports plain text and inline images.

### 🧮 Calculator
Full calculator with an optional **calculation history** log. Resizable and closeable without losing state.

### 🔍 Search Bar
Integrated search with support for:

`Google` · `Bing` · `DuckDuckGo` · `Brave` · `Ecosia` · `Startpage`

**Options include:**
- Live search suggestions
- Open results in current or new tab
- Custom placeholder text
- Adjustable bar width (240–680px)
- Compact mode for reduced padding
- One-click engine switching via the logo pill

## 📥 Import & Export

**Import options:**
- Pull from your browser's bookmark tree directly
- Restore from a previously exported ZenR `.json` file

**Export:**
- Save all pages, boards, and links as a portable `.json` backup

**Auto-Sync:**  
Enable *Auto-sync on new tab* to automatically re-import browser bookmarks every time a new tab opens.

## 🔒 Privacy

ZenR collects **zero** data. All bookmarks, settings, and wallpapers are stored locally in your browser using Firefox's standard `storage` API. The only external connections ZenR makes are ones you explicitly trigger:

- 🔍 A Pixabay search using **your own** API key
- 🌐 A search query submitted through the Search Bar widget
- 🔤 Google Fonts loaded at startup for font rendering

No analytics. No telemetry. No accounts. No servers.
