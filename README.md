## 🗂️ Layouts

### 🃏 Classic Boards
Draggable glass cards arranged in a free-form grid. Each board has its own name and optional color tint. Boards live inside named **Pages**, so you can keep Work, Entertainment, Research, and anything else cleanly separated.

### 🔲 Minimal Groups
A clean, distraction-free icon grid layout. Adjust links per row (3–16) and choose from three icon styles:

| Style | Look |
|---|---|
| Default | Solid favicon bubbles |
| Glass | Frosted glass icon chips |
| Minimal | Borderless favicon-only |

> **Note:** Links and widgets are independent between layouts and are not synced.

---

## 🪟 Liquid Glass Engine

Every card surface in ZenR uses a fully customizable glass renderer. All settings are live-editable from the Settings panel:

| Parameter | Range | Effect |
|---|---|---|
| Glass Blur | 0 – 72px | Frosted backdrop blur |
| Glass Density | 0 – 25% | Surface fill opacity |
| Saturation | 100 – 350% | Color vibrancy through glass |
| Brightness | 85 – 130% | Overall card brightness |
| Rim Glow | 0 – 100% | Inner edge light halo |
| Shadow Depth | 0 – 80% | Drop shadow intensity |
| Glass Tint | Color picker | Surface color overlay |
| Tint Intensity | 5 – 50% | Tint color strength |
| Corner Boost | 0 – 100% | Corner highlight effect |
| Edge Shimmer | 0 – 100% | Edge light animation |
| Refraction | 0 – 100% | Light bending distortion |

All values reset to defaults with one click.

## 🖼️ Wallpaper System

### 🎨 Gradients
One-click gradient presets — a curated palette always ready to go.

### 🏞️ Images
- **Pixabay Search** — Search millions of free images from inside ZenR using your own free Pixabay API key
- **Local Upload** — Upload JPG, PNG, WebP, or GIF files (multi-select supported)
- **Auto-rotate** — Get a fresh wallpaper on every new tab

### 🎬 Live Video Backgrounds
- **Pixabay Video Search** — Ambient video loops searchable directly inside ZenR
- **Local Upload** — Upload MP4, WebM, or OGV files
- **Auto-rotate** — Cycle your video collection on each tab open

### 🎛️ Background Controls
Fine-tune **overlay opacity** (0–90%) and **background blur** (0–30px) to keep content readable over any wallpaper.

## 🛠️ Local Installation (Chrome, Opera, Brave, Vivaldi, etc.)

For users on Chromium browsers where ZenR is not yet in the store, you can install it locally in Developer Mode in just a few steps:

1. **Download the Extension** — Click the **Download ZIP** button in the Downloads section above and extract the folder to your computer.
2. **Open Extensions** — In your browser, navigate to `chrome://extensions/` (or `opera://extensions/` / `brave://extensions/` / `vivaldi://extensions/`).
3. **Enable Developer Mode** — Toggle the **"Developer mode"** switch in the top-right corner of the page.
4. **Load the Extension** — Click **"Load unpacked"** in the top-left and select the extracted `ZenR-NewTab-main` folder.
5. **Open a New Tab** — ZenR will now replace your default new tab page. Enjoy!

> **Note:** When loading unpacked, your browser may show a reminder about developer mode extensions on startup — this is normal and safe to dismiss.

## 🔒 Privacy

ZenR collects **zero** data. All bookmarks, settings, and wallpapers are stored locally in your browser using the standard `storage` API. The only external connections ZenR makes are ones you explicitly trigger:

- 🔍 A Pixabay search using **your own** API key
- 🌐 A search query submitted through the Search Bar widget
- 🔤 Google Fonts loaded at startup for font rendering

No analytics. No telemetry. No accounts. No servers.
