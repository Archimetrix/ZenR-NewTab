## ✨ Key Features

* **🎨 Liquid Glass Theme:** A gorgeous, system-wide glassmorphism redesign of your new tab. Features beautifully rounded corners, blurred dynamic backgrounds, and custom color tints.
* **🔖 Advanced Bookmark Management:** Create custom pages, boards, and add your favorite links. Drag and drop links and boards to organize them exactly how you want.
* **🖼️ Dynamic Wallpapers:** Supports stunning gradient backgrounds, local image and video uploads, and built-in Pixabay integration for finding high-quality stock media directly within the extension.
* **🧮 Built-in Calculator:** A smart calculator widget with history support, floating directly on your new tab.
* **🕒 Customizable Clock:** Keep track of time with various stylish clock formats (Glass, Minimal, Neon, Retro, Bold), completely draggable and resizable.
* **🔍 Omnisearch Widget:** A beautiful search bar supporting multiple engines (Google, Bing, DuckDuckGo, Brave, Ecosia, Startpage) with live suggestions.
* **📝 Quick Notes:** A versatile note-taking widget. Add multiple notes, resize them, and even drag and drop images directly into the text.
* **📦 Import/Export & Sync:** Automatically syncs with your browser's native bookmarks. Easily export your setup as a JSON or HTML file, or import existing bookmarks.

## 🛠️ Local Installation (Chrome, Edge, Brave, Vivaldi etc.)

For users on Chromium browsers, you can install the latest stable build locally in Developer Mode in just a few seconds:

1.  **Download the Extension:** Click the **Download ZIP** button in the downloads section above and extract the folder to your computer.
2.  **Open Extensions:** Open your browser and type `chrome://extensions/` (or `edge://extensions/` / `brave://extensions/`) in the URL bar.
3.  **Enable Developer Mode:** Toggle the **"Developer mode"** switch in the top right corner of the page.
4.  **Load the Extension:** Click the **"Load unpacked"** button in the top left and select the extracted `ZenR-NewTab` folder.
5.  **Open a New Tab** and enjoy your ZenR workspace!

## 🖥️ Usage

* **Main Settings:** Click the gear icon in the top right corner of the new tab page to open the settings panel. From here, you can toggle widgets, change layouts (Classic vs Minimal Groups), adjust the Liquid Glass blur and tint, and manage your Pixabay API key.
* **Adding Content:** Use the "+" button on boards to add links. Right-click links or boards to edit or delete them. Use the top bar to add new pages.
